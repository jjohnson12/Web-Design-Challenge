Welcome

The purpose of this project was to analyze how weather changes as you move further away from the equator. To accomplish this analysis, we first generated a list of greater than 500 random cities from around the world using the python library Citipi. Next we accessed data from the OpenWeatherMap API to learn about the weather for these cities.


After assembling the dataset, we used Matplotlib to plot various aspects of the weather versus the latitude. Factors we studies including: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualization created from the analysis, as well as explanations and descriptions of any trends and correlations identified.
