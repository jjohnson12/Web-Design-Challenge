City Latitude versus Max Temperature

There is a strong correlation between the distance from the equator and the max temperature. This correlation is stronger in the Northern Hemisphere althought this is likely due to a greater number (and better distributed cities) in the Northern Hemisphere versus the Southern Hemisphere.

The purpose of this project was to analyze how weather changes as you move further away from the equator. You can learn more about the details behind this project at the home page
